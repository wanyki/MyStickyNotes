# MyStickyNotes

# 极简桌面便签 (MyStickyNotes) 📄

一款追求极致简洁、透明悬浮的 Windows 桌面便签工具。

## ✨ 功能特点
- **极致透明**：半透明黑色背景，完美融入任何桌面背景。
- **文字阴影**：自带文字黑阴影，无论壁纸深浅都清晰可见。
- **智能交互**：回车即固定，右键可重命名、复制、添加删除线或删除。
- **自动换行**：超长内容自动折叠，不占额外空间。
- **数据存档**：自动保存内容，关机重启记录不丢失。
- **系统托盘**：不在任务栏占位，双击右下角图标快速显隐，支持右键彻底退出。
- **自由缩放**：右下角手柄可随意调节窗口大小。

## 🚀 运行环境
- Windows 10/11
- .NET 8.0 Runtime

## 🛠️ 开发环境
- 语言：C#
- 框架：WPF (.NET 8)
- 开发工具：Visual Studio 2022

## 📦 如何安装
1. 前往 [Releases](../../releases) 页面下载最新的 `setup.exe`。
2. 运行安装程序，选择你的安装目录。
3. 桌面生成的快捷方式即可启动。
